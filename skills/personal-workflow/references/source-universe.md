# Source Universe

Use these sources in priority order when building an AI investment digest.

## Tier 1: Primary Sources

Check these first when relevant:

- OpenAI News: https://openai.com/news/
- Anthropic News: https://www.anthropic.com/news
- Google AI updates: https://blog.google/technology/ai/
- Meta AI blog: https://ai.meta.com/blog/
- NVIDIA Newsroom: https://nvidianews.nvidia.com/
- Microsoft official blogs: https://blogs.microsoft.com/
- AWS News Blog: https://aws.amazon.com/blogs/aws/
- GitHub Blog: https://github.blog/
- Hugging Face blog: https://huggingface.co/blog
- arXiv for major paper releases: https://arxiv.org/

## Tier 1 China: Primary Sources

Check these first for China-specific policy and platform developments:

- Cyberspace Administration of China: https://www.cac.gov.cn/
- Ministry of Industry and Information Technology: https://www.miit.gov.cn/
- Baidu AI and company announcements
- Alibaba Cloud and Alibaba official AI announcements
- Tencent and Hunyuan official announcements
- ByteDance and Volcano Engine official announcements
- SenseTime official announcements
- MiniMax official announcements
- Moonshot AI official announcements
- Zhipu AI official announcements

## Tier 2 China: Preferred Discovery Media

Use these actively for China AI discovery, then verify important items against
primary sources where possible:

- Jiqizhixin

## Tier 2 Global: High-Signal Media and Research Coverage

Use these to extend overseas coverage and interpretation:

- Reuters
- Bloomberg
- Financial Times
- The Information
- Semafor
- TechCrunch for startup and funding context

## Tier 3: Secondary Discovery Channels

Use selectively and verify before ranking highly:

- X or company founder posts
- GitHub trending repositories
- Hacker News discussion threads
- podcast summaries and newsletters
- aggregator sites and reposted articles

## Source Selection Rules

- Use Tier 1 first for factual verification.
- Use Tier 2 to broaden discovery and explain implications.
- Use Tier 3 only after verification or when monitoring early signals.
- If multiple sources conflict, prefer primary disclosure over media interpretation.
- If a source is paywalled, capture only the key confirmed facts and state limits.
- For China AI news, prioritize the named Chinese media above before broader generic tech coverage.
